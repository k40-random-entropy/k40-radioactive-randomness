# K‑40 Radioactive Randomness

This project explores the natural radioactivity of potassium‑40 (K‑40) as a source of physical randomness.  
Using household potassium compounds and a Geiger counter, the project documents measurement routines, statistical behavior, and the reproducibility of count‑based entropy.  
The goal is to create a transparent, empirical foundation for randomness derived from radioactive decay events.
The project treats each detected beta decay from potassium‑40 as a discrete physical event. 
These events form the basis for observing statistical behavior and deriving entropy from 
naturally occurring radioactive processes.


# K‑40 Radioaktive Zufälligkeit

Dieses Projekt untersucht die natürliche Radioaktivität von Kalium‑40 (K‑40) als Quelle physikalischer Zufallswerte.  
Mit haushaltsüblichen Kaliumverbindungen und einem Geigerzähler werden Messroutinen, statistische Eigenschaften und die Reproduzierbarkeit zählbasierter Entropie dokumentiert.  
Ziel ist eine transparente, empirische Grundlage für Zufallswerte, die aus radioaktiven Zerfallsereignissen gewonnen werden.
Das Projekt betrachtet jeden registrierten Beta‑Zerfall von Kalium‑40 als einzelnes physikalisches 
Ereignis. Diese Ereignisse bilden die Grundlage für die Beobachtung statistischer Eigenschaften 
und die Ableitung von Entropie aus natürlich vorkommenden radioaktiven Prozessen.
